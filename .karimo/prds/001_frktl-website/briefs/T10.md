# Brief: T10 — CTA Section — "Get in Touch" with mailto + Investor Pitch Context

**Wave:** 3 | **Complexity:** 1/10 | **Priority:** Must | **Depends on:** T02

---

## Context

The current `frktl-explainer.jsx` ends with a minimal footer (FRKTL logo + "Energizing the Globe."). There is no call-to-action section. This task adds a dedicated CTA section before the footer that converts investor intent into direct contact.

The primary action is a mailto link to `eduardo@frktlpower.com`. No JavaScript email obfuscation — a plain `<a href="mailto:...">` link. The secondary action is a visible email address as a plain text link.

This is a Wave 3 task (parallel with T06, T07, T08) that depends only on T02 (for section data types and CSS tokens). It is the smallest and most straightforward task in the PRD.

---

## Design Direction

The CTA section is a visual departure from the dark navy palette of the content sections. It uses the warm orange/cream editorial palette to signal "this is different — this is the action point."

**Visual language:**
- Background: warm dark with orange/cream gradient overlay, or a direct `--color-dark-mid` with strong orange accents
- `AtomicStarburst` decoration in the background (T08 creates this component — if not yet available, use a simple CSS starburst or skip)
- Section eyebrow: "READY TO POWER THE FUTURE?" in Atomic Age font, orange color
- Headline: "Get in touch." in Instrument Serif at large scale
- Body: investor pitch context (copy specified below — use exactly)
- CTA button: bold, teal-to-cyan gradient background, white text, rounded, generous padding
- Hover state: slight scale up (`whileHover={{ scale: 1.04 }}`) + brightness increase via `hover:brightness-110` Tailwind class
- Secondary line: plain email link below the button

**Mobile:** Single column, full-width button. Heading scales down with clamp(). No horizontal overflow.
