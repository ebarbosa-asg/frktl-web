# Brief: T05 — FRKTL Wordmark — Custom CSS/SVG Logotype

**Wave:** 2 | **Complexity:** 2/10 | **Priority:** Must | **Depends on:** T01

---

## Context

No FRKTL logo exists. This task designs and implements one from scratch as a React SVG component. The wordmark must be self-contained inline SVG paths — no external font files, no `<text>` elements with font-family dependencies. It must scale cleanly from 80px to 240px wide without pixelation and work in three color variants.

The wordmark will be used by T06 (sticky header), T08 (chapter markers), and the footer. It is a critical brand asset for the investor demo at frktlpower.com.

**Design philosophy:** "FRKTL" is vowel-stripped — industrial, compressed, efficient. The wordmark should feel technical and confident. The PRD instructs: "Go all out." This is not a simple text render — it's a designed letterform treatment with a nuclear/energy accent.

---

## Design Direction

The 50s atomic-age layer is one of three aesthetic registers in the site. The wordmark lives primarily in the "modern tech" register but with subtle energy/nuclear references. Avoid being too literal (no uranium symbols, no radiation trefoils). Reference: Life Magazine nuclear feature spreads from 1955, bold condensed display typography.

### Recommended approach (use your design judgment)

**Option A — Geometric condensed with hexagon accent (recommended):**
- "FRKTL" in tight, bold condensed custom letterforms
- A hexagonal atom mark to the left of or above the letters
- The hexagon references the prismatic graphite fuel block geometry (the core building block of the reactor)
- The inner hexagon has a small circle or dot at center (referencing the UCO kernel inside TRISO)
- Letter spacing: slightly negative tracking for compressed industrial feel
- The F could have a distinctive treatment — horizontal bars suggesting energy levels

**Option B — Space Mono with custom treatment:**
- "FRKTL" in Space Mono (the existing monospace font) at bold weight
- But: Space Mono paths embedded as SVG for true self-containment
- Add: a small atomic orbit ellipse as a decorative element
- This is a lower-effort but potentially less distinctive option

**Option C — Atom orbit prefix:**
- A small SVG atom icon (nucleus + orbit ellipses) to the left
- Followed by "FRKTL" in a custom bold condensed face
- Clean, startup-credible, nuclear without being kitschy

The component should implement **at minimum the letters as `<path>` elements** — either custom-drawn letterforms or traced/outlined from a condensed sans-serif. Simple geometric construction is acceptable (letters built from rectangles, arcs, and lines).

### Color variants

| Prop value | Foreground | Use case |
|-----------|-----------|----------|
| `"light"` (default) | White (#e8f4f8) | Dark navy header, footer |
| `"orange"` | Atomic orange (#e8521a) | Chapter markers, editorial accents (T08) |
| `"dark"` | Dark (#0a1628) | Light backgrounds, print |

## Technical Specification

### File: components/ui/FRKTLWordmark.tsx

```tsx
"use client";

interface FRKTLWordmarkProps {
  variant?: "light" | "orange" | "dark";
  size?: number; // Width in pixels, default 160
  className?: string;
}

const VARIANT_COLORS = {
  light: "#e8f4f8",
  orange: "#e8521a",
  dark: "#0a1628",
} as const;

export function FRKTLWordmark({
  variant = "light",
  size = 160,
  className,
}: FRKTLWordmarkProps) {
  const color = VARIANT_COLORS[variant];
  const aspectRatio = 4; // width:height ratio of the wordmark — adjust to your design
  const height = size / aspectRatio;

  return (
    <svg
      viewBox="0 0 160 40"  // Adjust viewBox to match your letterform design
      width={size}
      height={height}
      className={className}
      aria-label="FRKTL Energy"
      role="img"
    >
      {/* Hexagon mark */}
      <polygon
        points="8,20 14,11 26,11 32,20 26,29 14,29"
        fill="none"
        stroke={color}
        strokeWidth="2.5"
        strokeLinejoin="round"
      />
      <circle cx="20" cy="20" r="3.5" fill={color} />

      {/* FRKTL letterforms — implement as <path> elements */}
      {/* F */}
      <path d="M 42 10 L 42 30 M 42 10 L 56 10 M 42 19 L 54 19" stroke={color} strokeWidth="3.5" strokeLinecap="square" fill="none" />
      {/* R */}
      <path d="M 62 10 L 62 30 M 62 10 L 72 10 Q 77 10 77 16 Q 77 22 72 22 L 62 22 M 70 22 L 77 30" stroke={color} strokeWidth="3.5" strokeLinecap="square" fill="none" />
      {/* K */}
      <path d="M 83 10 L 83 30 M 83 20 L 94 10 M 85 18 L 94 30" stroke={color} strokeWidth="3.5" strokeLinecap="square" fill="none" />
      {/* T */}
      <path d="M 100 10 L 114 10 M 107 10 L 107 30" stroke={color} strokeWidth="3.5" strokeLinecap="square" fill="none" />
      {/* L */}
      <path d="M 120 10 L 120 30 L 133 30" stroke={color} strokeWidth="3.5" strokeLinecap="square" fill="none" />
    </svg>
  );
}

export default FRKTLWordmark;
```

**Important:** The letterform paths above are a starting template using simple stroked geometry. The agent executing this task has full design latitude to improve the letterforms. Options:
- Use filled `<path>` elements instead of stroked outlines for a bolder, more logotype feel
- Adjust proportions, letter spacing, and the hexagon mark position
- Add subtle details like a small triangular notch or fission-crack on the F
- The template above uses construction-line style (stroked) — filled forms will look more finished

### Accessibility

The SVG must have `role="img"` and `aria-label="FRKTL Energy"` on the root element. No `aria-hidden` — this is the brand identity, not decorative illustration.

### Scaling test

The component must look good at these sizes:
- `size={80}` — mobile header
- `size={120}` — desktop header
- `size={160}` — hero / default
- `size={240}` — footer / large display

The `viewBox` should be fixed; only `width` and `height` attributes scale. This is standard SVG scaling behavior.

### Contrast check

White variant (`#e8f4f8`) on `--color-dark` (`#0a1628`) must achieve 4.5:1 contrast ratio for WCAG AA compliance. White on near-black always passes — confirm visually.
