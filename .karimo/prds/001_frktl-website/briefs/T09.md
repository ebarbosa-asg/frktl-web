# Brief: T09 — Spline 3D Isometric Hero Scene (Cute Nuclear Factory, Lazy-Loaded)

**Wave:** 2 | **Complexity:** 4/10 | **Priority:** Must | **Depends on:** T01

---

## Context

The hero section needs a 3D isometric scene to anchor the above-the-fold experience and deliver the "wow" moment for investors viewing frktlpower.com for the first time. The scene is a cute toy-town nuclear factory — friendly, not ominous.

This task runs in Wave 2 (parallel with T02, T03, T04, T05) and depends only on T01. It creates two components: `HeroScene.tsx` (the Spline wrapper) and `HeroSceneFallback.tsx` (the static SVG that is the actual LCP element).

**Performance constraint:** The Spline scene must NOT block LCP. The static SVG fallback renders immediately and is the LCP target. Spline loads lazily after the fallback paints. On mobile (< 768px), only the static fallback is shown — no Spline at all.

**Scene URL:** The Spline scene URL will be a placeholder (`https://prod.spline.design/PLACEHOLDER-SCENE-URL/scene.splinecode`). The user (Eduardo) will replace this with the real scene URL after designing the scene in Spline's editor. The component architecture must make this substitution trivial (a single constant at the top of the file).

## Design Direction

### Scene aesthetic: "Toy Town Nuclear Factory"

The Spline 3D scene (when created) should show:
- 1–2 small modular reactor buildings in isometric projection — low-poly, rounded corners, terracotta/warm green palette
- A cooling tower with soft white puff cloud smoke rising
- A miniature flatbed delivery truck to the left of the factory
- A small ground plane with subtle grass/green texture
- Warm golden-hour lighting — sun coming from upper right
- Human-scale tiny workers near the building entrance (optional but delightful)

**Palette:** Warm greens (`#5a8a4a`), terracotta (`#c4714a`), powder blue (`#89c0d4`), cream (`#f5e6c8`). The scene should feel like it belongs in a well-funded board game, not an industrial facility.

**Projection:** Isometric (orthographic camera at 45° rotation, 35.26° elevation). This matches the Dribbble references (Nuclear Powerplant shot at dribbble.com/shots/2769670).

### Static SVG fallback: Simplified isometric illustration

The fallback SVG (rendered immediately as LCP) must convey the same "toy town factory" concept in 2D SVG. Design guidance:
- Isometric projection using SVG transforms and careful coordinate positioning
- Three main shapes: a reactor building (isometric box), a cooling tower (trapezoid with puff circle), a small truck
- Colors match the Spline scene palette: terracotta building, warm green ground, powder blue sky band
- Simple, clean, "illustration style" — not a technical diagram

### File: components/HeroScene.tsx

```tsx
"use client";

import dynamic from "next/dynamic";
import { Suspense } from "react";
import { HeroSceneFallback } from "./HeroSceneFallback";

// Scene URL — replace with real Spline scene URL when available
const SPLINE_SCENE_URL =
  "https://prod.spline.design/PLACEHOLDER-SCENE-URL/scene.splinecode";

// Lazy-load Spline — never on mobile, never blocks LCP
const SplineScene = dynamic(
  () => import("@splinetool/react-spline/next"),
  {
    ssr: false,  // Never server-render Spline (it requires browser APIs)
    loading: () => <HeroSceneFallback />,  // Show fallback while loading
  }
);

interface HeroSceneProps {
  className?: string;
}

function isMobile(): boolean {
  if (typeof window === "undefined") return false;
  return window.innerWidth < 768;
}

export function HeroScene({ className }: HeroSceneProps) {
  // On mobile: always use static fallback (Spline is heavy on mobile GPU)
  const useFallback = typeof window !== "undefined" && isMobile();

  return (
    <div
      className={className}
      style={{
        width: "100%",
        height: "100%",
        minHeight: 400,
        position: "relative",
      }}
    >
      {useFallback ? (
        <HeroSceneFallback className="w-full h-full" />
      ) : (
        <Suspense fallback={<HeroSceneFallback className="w-full h-full" />}>
          <SplineScene
            scene={SPLINE_SCENE_URL}
            style={{ width: "100%", height: "100%" }}
          />
        </Suspense>
      )}
    </div>
  );
}
```

**Alternative mobile detection using CSS media query** (more reliable than JS window.innerWidth):

```tsx
// Prefer CSS-based approach over JS isMobile()
// Use Tailwind responsive classes or CSS media query
<div className="block md:hidden">
  <HeroSceneFallback />
</div>
<div className="hidden md:block">
  <Suspense fallback={<HeroSceneFallback />}>
    <SplineScene scene={SPLINE_SCENE_URL} style={{ width: "100%", height: "100%" }} />
  </Suspense>
</div>
```

The CSS approach is preferred because it doesn't depend on JS execution order and works correctly during SSR.
